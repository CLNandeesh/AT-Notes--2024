Invoking Browser
================
Ex:
public class BrowserOperation1 {
    public static void main(String[] args) {

        WebDriver driver = new ChromeDriver();
        driver.get("https://demo.opencart.com/");
        String title = driver.getTitle();
        System.out.println("Title is: " + title);
        driver.quit();

    }
}
